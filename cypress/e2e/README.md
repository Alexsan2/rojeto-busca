# 🧪 Automação de Testes com Cypress

Este projeto faz parte do meu portfólio de **QA Automation**. Aqui realizo testes de ponta a ponta (E2E) utilizando o framework Cypress para validar funcionalidades de aplicações web.

## 🚀 Tecnologias Utilizadas
* [Cypress](https://www.cypress.io/)
* JavaScript / Node.js
* Mochawesome (Relatórios)

## 📋 Funcionalidades Testadas
* **Login:** Sucesso e erro com usuário bloqueado.
* **Carrinho:** Adição de produtos e validação de contagem.
