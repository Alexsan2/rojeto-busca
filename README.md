# 🧪 Projeto de Automação: Busca e E-commerce

Este repositório contém a automação de testes para o site **Sauce Demo**, utilizando o framework **Cypress**. O objetivo é validar fluxos críticos como login e gerenciamento de carrinho, garantindo a qualidade da aplicação.

## 🚀 Tecnologias Utilizadas
* **Framework:** [Cypress](https://www.cypress.io/)
* **Linguagem:** JavaScript
* **Relatórios:** Mochawesome (HTML Reports)
* **Node.js:** Gerenciador de pacotes (npm)

## 📋 Cenários de Teste (E2E)
Atualmente, o projeto cobre os seguintes cenários:
1.  **Login com sucesso:** Valida o acesso com usuário padrão.
2.  **Login com erro:** Valida a mensagem de erro para usuário bloqueado.
3.  **Fluxo de Carrinho:** Adição de produtos e validação do contador de itens.
